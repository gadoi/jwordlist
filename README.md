jwordlist
====================

About
-----
- This tool will generate a wordlist based on list of word that you import.

Installation
------------
- `$ git clone https://github.com/j3ssie/jwordlist.git `
- `$ cd jwordlist/`
- `$ python3 jwordlist.py`
